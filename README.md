### 🚀 Key Project Spotlight

#### **SkillBridge | Hyperlocal Hiring Marketplace**
*A production-ready platform connecting local talent using real-time geospatial intelligence.*

- **Geospatial Optimization**: Leveraging MongoDB `2dsphere` for hyperlocal job discovery.
- **Real-time Pipeline**: Event-driven architecture with `Socket.IO` for live application tracking.
- **AI Integration**: High-accuracy `NLP` voice registration system via Groq APIs.
- **Modern UI**: 60FPS glassmorphic interface powered by `Framer Motion` and `Tailwind CSS`.
